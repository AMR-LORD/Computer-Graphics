### **๐ Random-Scan Displays**


- **It uses an electron beam which operates like a pencil to create a line image on a CRT screen.**
- **The picture is constructed out of a sequence of straight-line segments.**
- **Each line segment is drawn on the screen by directing the beam to move from one point on the screen to the next, where its x & y coordinates define each point.**
- **After drawing the picture, the system cycles back to the first line and redraws all the lines of the image.**
- **Random scan monitors are also known as vector displays, stroke-writing displays, or calligraphic displays.**
